# 도구 선택 가이드

## GitHub CLI (gh)
- GitHub 저장소 작업이 필요할 때
- 이슈나 PR을 CLI에서 관리하고 싶을 때
- GitHub Actions 워크플로우를 실행하거나 확인할 때
- 웹 브라우저 없이 GitHub 작업을 자동화할 때
- CI/CD 파이프라인에서 GitHub API를 호출할 때

## Terraform Resource 저장소
- AWS 인프라 리소스를 생성/수정/삭제할 때
- ECR, S3, DynamoDB, IAM 등 AWS 서비스 설정이 필요할 때
- Kafka(Confluent), Datadog, PagerDuty 설정을 변경할 때
- 인프라 변경을 코드로 관리하고 버전 관리가 필요할 때
- Atlantis를 통해 안전하게 인프라를 배포할 때
